Go to the [Mail Server](/mail-servers) page and add your server by following these steps:

## Steps:

<Steps>
### Sign up/ Login to your Freesend account

### Navigate to the Mail Servers section in the Navbar

<img
  className="w-[200px]"
  src="/images/mail-server-navigation.png"
  alt="Mail Server Navigation"
/>

### Add a new mail server by clicking on "Add Mail Server"

### Fill in the details of your SMTP Server

<img
  className="w-[200px]"
  src="/images/add-mail-server-details.png"
  alt="Add mail server details"
/>

### Done!

You can now [Create an API Key](/docs/configuration/api-keys) to send emails from this SMTP Server!

</Steps>
